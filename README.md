## Getting Started

```
# clone repo
git clone https://github.com/MaximilianHagelstam/larkanscran.git

# move into project
cd larkanscran
```

```
# Server

# move into folder
cd server

# install dependencies
npm install

# start server
npm run dev
```

```
# Client

# move into folder
cd client

# create .env
cp .env.example .env

# install dependencies
npm install

# run app
npm start
```
